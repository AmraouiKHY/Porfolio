# Khireddine Amraoui - Portfolio

Welcome to my personal portfolio website! 
This portfolio website is built with Next.js, TypeScript, Framer Motion, and Tailwind CSS. I designed it to be fast, responsive, and visually appealing while highlighting my experience in web development and data science.

## Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS, Framer Motion
- **Data**: Static data architecture for optimal performance
- **Styling**: Custom Tailwind CSS with smooth animations
- **Deployment**: Optimized for modern hosting platforms

## Development Setup

To run this portfolio locally:

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the portfolio.

## What You'll Find Here

- **About Me**: My background, skills, and professional journey
- **Education**: My academic achievements and certifications
- **Experience**: Professional work history and key projects
- **Projects**: Showcase of my development work and data science projects
- **Skills**: Technical proficiencies across various technologies
- **Contact**: Ways to get in touch with me

## Key Features

- **Responsive Design**: Fully optimized for all device sizes
- **Smooth Animations**: Engaging transitions and hover effects
- **Fast Performance**: Static generation for lightning-fast loading
- **SEO Optimized**: Structured for search engine visibility
- **Modern UI/UX**: Clean, professional design aesthetic


## Deployment

This portfolio is deployed on Vercel for optimal performance and reliability. The deployment process is automated through GitHub integration, ensuring the live site is always up-to-date with the latest changes.
